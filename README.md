## About

Hi, I'm **Anshul Rajpal** — a Full Stack Developer passionate about building modern web experiences.

I work primarily with **React, JavaScript, Supabase, and modern web technologies**, focusing on developing responsive, interactive, and user-focused web applications.

My development interests extend beyond functionality. I enjoy working with **UI/UX, animations, responsive interfaces, and modern web design**, combining engineering with thoughtful visual experiences.

On the backend, I work with **Node.js and Supabase**, while using **PostgreSQL** for structured application data.

I am continuously learning new technologies, improving my development workflow, and exploring advanced web development concepts.
